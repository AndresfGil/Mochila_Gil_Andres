Inicia un repositorio local
git init

Clonar un repositorio existente
git clone <https://link-con-nombre-del-repositorio>

Este comando creará una rama en local
git branch <nombre-de-la-rama>

Usaremos git checkout principalmente para cambiarte de una rama a otra
git checkout <nombre-de-la-rama>

Este comando nos da toda la información necesaria sobre la rama actual
git status

Usamos este comando para incluir los cambios del o de los archivos en tu siguiente commit
git add <archivo>

Este comando nos permite guardar los cambios realizados hasta ese momento
git commit -m "mensaje de confirmacion"

Este comando envía tus commits al repositorio remoto
git push <nombre-remoto> <nombre-de-tu-rama>

Se utiliza para recibir actualizaciones del repositorio remoto
git pull <nombre-remoto>
